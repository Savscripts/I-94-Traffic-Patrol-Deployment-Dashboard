# I-94-Traffic-Patrol-Deployment-Dashboard
Tableau dashboard for optimizing police patrol deployment on I-94 highway based on traffic volume, time of day, weather, and holiday patterns.

## Overview
An operational Tableau dashboard designed to help law enforcement agencies optimize patrol staffing on Interstate 94 based on real traffic patterns, time-of-day surges, weather conditions, and holiday schedules.

## Key Insights
- Peak traffic hits **7,280 vehicles** (max) with an average of **3,260 vehicles** per scenario
- **4 PM is the highest-risk hour** — traffic surges sharply during evening rush (hours 15–17)
- Weekday **morning hours (6–9 AM)** trigger Priority 1 (All-Hands) deployment across Mon–Fri
- **Non-holiday days** see 3x more traffic (~3,263) than most observed holidays (~519–1,108)
- Weather has minimal impact on volume — all conditions hover near the 3,161 average

## Features
- Weekly Deployment Map: hour-by-hour patrol priority grid (Priority 1 / 2 / 3) across all 7 days
- Daily Traffic Pulse chart identifying surge windows and minimum staffing periods
- Holiday Traffic analysis to identify low-volume staffing reduction opportunities
- Weather influence chart to assess road condition impact on volume
- Filters for Weather Category and Day of Week

## Tools Used
- Tableau Desktop / Tableau Public
- Dataset: I-94 Interstate Highway Traffic Volume
